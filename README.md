# Sistemas-Cadastro-de-ALunos# Sistema de Cadastro de Alunos

Sistema desktop em Python com interface gráfica (tkinter) para
cadastro e gerenciamento de notas de alunos.

## Funcionalidades
- Login com perfis: professor e aluno
- Professor: cadastrar, excluir e ver todos os alunos
- Aluno: visualizar apenas suas próprias notas
- Dados salvos automaticamente em planilha Excel

## Como executar
1. Instale as dependências:
   pip install -r requirements.txt

2. Execute o sistema:
   python sistema_alunos.py
